# Technical Documentation: Southeast Asia XCO2 Reconstruction

## 1. Project Overview
This project aims to produce a **0.1° monthly seamless XCO2 dataset** (2015–2024).

## 2. Algorithm (STK)
We use a **Residual-corrected Spatio-Temporal Kriging** method based on:
- **OCO-2** Satellite Level 2 data
- **CAMS** Model background fields

## 3. Data Structure
- Spatial Resolution: 0.1 degree
- Region: South China and SE Asia
